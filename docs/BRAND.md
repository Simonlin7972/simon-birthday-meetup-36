# Simon's Party — Brand Guidelines

> Simon 的 36 歲生日 & 告別派對 · 視覺識別指南
> 版本 0.1（依兩張主視覺整理；色票為目視取色的近似值，之後可微調）

---

## 1. 品牌個性 Brand Personality

溫暖、手繪、像收到一張用心的邀請卡。不華麗、不冷調，而是**親近、有溫度、帶點玩心**。

| 關鍵字 | 說明 |
|--------|------|
| 溫暖 Warm | 牛皮紙底色、奶油白、暖棕，整體偏暖、低彩度 |
| 手感 Handmade | 粗黑描邊的插畫、紙質紋理、手寫感波浪線 |
| 玩心 Playful | 彩旗、彩帶、氣球、愛心、`YA!`、揮手的角色 |
| 親近 Friendly | 圓潤的字體與圖形、沒有銳利邊角 |
| 誠摯 Sincere | 留白、字距拉開的標題，安靜而慎重的邀請感 |

一句話：**「像牛皮紙信封裡，一張手繪的生日邀請卡。」**

> ⚠️ 注意：目前 app 的實作仍是「暗色 + 金色」的舊風格。這份文件描述的是**新的主視覺方向**，未來改版會是一次完整的 reskin（深色 → 暖色紙感）。

## 2. 色彩系統 Color Palette

所有色票為目視近似值，建議之後用主視覺原檔重新取色校準。命名採「語意 + 描述」。

### 中性色 — 背景與紙感 Neutrals

| Token | Hex | 用途 |
|-------|-----|------|
| `sand-bg` | `#D9C4AC` | **主背景**，牛皮紙暖沙底 |
| `sand-deep` | `#CDB59A` | 較深的沙，分隔／陰影底 |
| `paper` | `#EDE0CF` | 紙卡、次級面 |
| `cream` | `#F6EEE1` | 卡片、淺色面 |
| `white` | `#FCFAF6` | 純白（角色衣服、高光） |

### 主棕 — 文字與展示 Brown Ink

| Token | Hex | 用途 |
|-------|-----|------|
| `cocoa` | `#6E4A2E` | **主文字色 / 展示大標**（`Happy`、`快樂`） |
| `espresso` | `#4A3322` | 深棕，強調、最高層級文字 |
| `mocha` | `#A8826A` | 中棕（領子、鈕扣、次級文字） |

### 橘 — 主要強調色 Orange (Primary Accent)

| Token | Hex | 用途 |
|-------|-----|------|
| `orange` | `#E88B3A` | **主強調色**（毛帽、褲子、`Party!`、CTA） |
| `orange-soft` | `#E7A86E` | 柔橘（愛心、hover、次級強調） |
| `apricot` | `#F2C6A0` | 杏桃，淺色點綴／底 |

### 次要色 Secondary

| Token | Hex | 用途 |
|-------|-----|------|
| `mauve` | `#B59C92` | 霧灰玫瑰（`Welcome to Simon's` 標題） |
| `slate` | `#8B95A3` | 霧藍（`誠摯邀請你` 邀請字、安靜的次要資訊） |

### 粉彩點綴 — 彩旗／彩帶 Pastel Accents

僅用於裝飾性元素（彩旗、confetti、tag），不用於大面積。

| Token | Hex |
|-------|-----|
| `pastel-yellow` | `#F2CE7E` |
| `pastel-green` | `#BBD08A` |
| `pastel-blue` | `#A9CBE0` |
| `pastel-peach` | `#F4C7A2` |

### 描線 Line

| Token | Hex | 用途 |
|-------|-----|------|
| `ink` | `#1E1A17` | 插畫粗黑描邊（**僅用於插畫**，不建議當大面積文字色） |

### 配色原則

- **6 成中性、3 成棕、1 成橘**：背景與留白以暖沙／奶油為主，棕色承載文字，橘色只在重點處點睛。
- **低彩度為基調**：除了橘與彩旗，其餘顏色都偏霧、偏柔。
- **粉彩只做裝飾**：彩旗那組顏色不要拿來填大色塊或當文字色。
- **對比**：深棕（`cocoa`/`espresso`）配暖沙背景即可達到足夠可讀性；橘色文字盡量加大、加粗使用。

## 3. 字體系統 Typography

主視覺呈現的是**圓潤、飽滿、友善**的字體個性。

### 目前實作

| 角色 | 字體 | 說明 |
|------|------|------|
| 中文 | **ChironGoRoundTC**（圓體） | 與主視覺的圓潤中文一致 ✅ |
| 英文 / 數字 | **Archivo Black** | 粗黑、力道強 |

### 與主視覺的落差（待你決定）

主視覺裡的英文（`Welcome to`、`Birthday Party!`）是**圓潤的粗體**，而 Archivo Black 是方正的粗黑體。兩個方向都成立：

- 想**完全貼合**主視覺的可愛圓潤感 → 英文可考慮換成圓體 display（如 Varela Round / Baloo 2 / Quicksand Bold 一類）。
- 想要**對比與份量感** → 維持 Archivo Black，靠它的厚重做視覺重點。

> 這題留給你拍板，brand 文件先記錄兩者差異。

### 排版手法（從主視覺歸納）

- **展示大標**：超粗、字級大、可微微傾斜或弧線排列（`Happy`、`快樂`、`YA!`）。
- **邀請／標籤**：字距大幅拉開（letter-spacing），營造安靜慎重感（`Simonの生日小聚`、`誠摯邀請你`）。中文標題逐字拉開間距是這個品牌的招牌手法。
- **層級用顏色區分**：同一行可混用 `mauve` + `orange`（如 `Birthday` 灰、`Party!` 橘）。

---

## 4. 插畫風格 Illustration Style

- **粗黑描邊**（`ink`）+ **平塗色塊**，無漸層、無寫實陰影。
- **圓角、圓潤造型**，肢體與物件都是圓滑線條。
- 主角：戴橘色毛帽、圓框眼鏡、白上衣棕領、橘長褲、抱著筆電、揮手的人物（= Simon 的化身）。
- 手部、表情誇張可愛，帶手繪不完美感。

**點綴元素庫**：彩旗（bunting）、彩帶 popper、氣球（線稿）、愛心、手寫波浪底線、`YA!`、驚嘆號。

---

## 5. 材質與表面 Texture & Surfaces

- **牛皮紙紋理**：背景帶細微的紙張折痕／纖維質感（banner 可見），是這個品牌的重要肌理。
- **圓角卡片**：介面元件沿用大圓角（與插畫圓潤呼應）。
- **貼紙感**：部分裝飾元素帶白色描邊外框，像貼紙貼上去（banner 的彩帶、氣球）。

---

## 6. 語氣 Voice & Tone

- 語言：**繁體中文為主**，可混搭輕鬆英文（`Welcome`、`YA!`）與日文助詞（`Simonの生日小聚`）。
- 口吻：溫暖、口語、像朋友說話；慎重的邀請語（`誠摯邀請你`）與俏皮語（`YA!`）並存。
- 用 emoji 適度增添溫度，但展示性文案保持乾淨。

---

## 7. Do / Don't

**Do**
- ✅ 用暖沙／奶油當底，大量留白
- ✅ 橘色只點睛、深棕承載文字
- ✅ 標題字距拉開，營造邀請卡的慎重感
- ✅ 插畫一律粗黑描邊 + 平塗

**Don't**
- ❌ 用純白或冷灰當主背景（會失去紙感溫度）
- ❌ 大面積使用粉彩或橘色
- ❌ 插畫加漸層、寫實陰影或細線描邊
- ❌ 把 `ink` 純黑拿來當大段內文色（改用 `cocoa`）

## 8. 對應到 App 的設計 Token（給未來改版用）

把上面的色彩整理成可直接落地的 CSS 變數草案，未來改版時可替換 `src/styles/global.css` 的 `:root`。**目前尚未套用**，僅作參考。

```css
:root{
  /* 中性 / 紙感 */
  --sand-bg:#D9C4AC; --sand-deep:#CDB59A;
  --paper:#EDE0CF;   --cream:#F6EEE1;   --white:#FCFAF6;
  /* 主棕（文字 / 展示） */
  --cocoa:#6E4A2E;   --espresso:#4A3322; --mocha:#A8826A;
  /* 橘（強調） */
  --orange:#E88B3A;  --orange-soft:#E7A86E; --apricot:#F2C6A0;
  /* 次要 */
  --mauve:#B59C92;   --slate:#8B95A3;
  /* 粉彩點綴 */
  --pastel-yellow:#F2CE7E; --pastel-green:#BBD08A;
  --pastel-blue:#A9CBE0;   --pastel-peach:#F4C7A2;
  /* 描線 */
  --ink:#1E1A17;

  /* 語意對應（建議改版時用語意 token，不要直接用顏色名） */
  --bg:var(--sand-bg);
  --surface:var(--cream);
  --text:var(--cocoa);
  --text-strong:var(--espresso);
  --text-muted:var(--mocha);
  --accent:var(--orange);
  --line:rgba(30,26,23,.12);
}
```

> 改版方向預覽：背景 `#0d0c10`（黑）→ `#D9C4AC`（暖沙）；強調 `#e7b667`（金）→ `#E88B3A`（橘）；文字 `#f3ede2`（米白）→ `#6E4A2E`（棕）。這是一次「深色奢華」→「暖色紙感」的整體翻轉。
